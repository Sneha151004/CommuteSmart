# CommuteSmart
=======
<h1>Features</h1>
<h2>Carpoling</h2>
<ul>
<h3>Give a Ride</h3>
  <li>Users can offer carpool rides by posting available seats along with details such as the route and number of seats. This helps others in the community find eco-friendly travel options, and the carpool host earns eco-coins for providing a ride.</li>
  </ul>
<ul>
<h3>Ask for a Ride</h3>
  <li>Users seeking a ride can browse through available carpool options based on location preferences. They can request to join a ride, and once accepted by the host, they earn eco-coins for commuting in an eco-friendly way.</li>
  </ul>

<h2>Leaderboard</h2>
<ul>
<li>A leaderboard tracks the users who have earned the most eco-coins through eco-friendly commuting behaviors, such as carpooling or participating in campaigns. This fosters a competitive yet fun way to motivate sustainable practices.</li>
</ul>

<h2>Reward</h2>
<ul>
 <li>Users can redeem their accumulated eco-coins by purchasing goodies through the appâ€™s reward store.The eco-coins encourage continuous participation in sustainable commuting practices.</li>
</ul>

<h2>Community Section</h2>
<ul>
  <li>A dedicated community space where users can interact, share experiences. The community encourages collaboration and promotes eco-friendly initiatives across the user base.</li>
<h3>Host a Campaign</h3>
  <li>Users can create campaigns aimed at raising awareness and promoting eco-friendly commuting behaviors and earn eco-coins. Campaigns can set specific goals, such as reducing carbon emissions or encouraging more carpooling.</li>
  </ul>
<ul>
<h3>Join a Campaign</h3>
  <li>Users can join existing campaigns that align with their eco-friendly goals. By participating, they contribute to the campaign objectives and earn eco-coins.
</li>
  </ul>
